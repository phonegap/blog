The Notification API allows for visual, audio, and tactile notifications. In this post I want to focus on the visual notifications and how they differ from the built in web view notifications.To begin, I created a simple HTML interface with buttons that would let me test alerts, confirms, and prompts. These are the three forms of visual notifications both in vanilla JavaScript as well as PhoneGap's Notification API.
